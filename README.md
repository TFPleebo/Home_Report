# House Power

**Live dashboard: https://tfpleebo.github.io/Home_Report/**

Personal household energy dashboard for a Cape Cod in eastern Ohio. Self-contained
HTML — daily electric cost vs. a weather-based model, live room temperatures,
a cost scenario simulator, and a battery outage planner.

Data updates are pushed automatically each morning by a scheduled job.
This is a personal reference tool; the interesting part is the single-file,
zero-dependency approach: all data is embedded at publish time.
